M365 Build Script Specifications (Programmer Reference)

These specifications describe the following build scripts:
30-Set-ExoMailboxRegionalConfiguration.ps1
31-Set-ExoMailboxEmailAddresses.ps1 40-Create-SPOSites.ps1
41-Set-SPOSiteAdmins.ps1 42-Create-SPOHubSites.ps1
43-Associate-SPOSitesToHub.ps1 52-Add-TeamOwners.ps1
93-Compare-DesiredVsActual.ps1

Global Design Expectations

-   PowerShell 7.x
-   CSV-driven input with required header validation
-   Module presence and version checks
-   Auto-connect to required services (Graph, EXO, SPO)
-   Per-record error handling
-   Native -WhatIf support
-   Timestamped results CSV output
-   Retry with exponential backoff for transient failures
-   Idempotent where possible

Standard Output Columns (recommended)

TimestampUtc TargetKey TargetType Action Success ErrorCode ErrorMessage
ResultSummary

30-Set-ExoMailboxRegionalConfiguration.ps1

Purpose: Set mailbox regional settings (language, timezone, date/time
format) for targeted mailboxes.

Input CSV Required Headers: MailboxIdentity Language TimeZone DateFormat
TimeFormat LocalizeDefaultFolderName (optional)

Behavior: - Resolve mailbox by UPN or primary SMTP. - Apply regional
configuration settings. - Support idempotency. - Log before/after
summary in output.

31-Set-ExoMailboxEmailAddresses.ps1

Purpose: Add/remove proxy addresses and optionally set primary SMTP
address.

Input CSV Required Headers: MailboxIdentity PrimarySmtpAddress
(optional) AddAliases (semicolon-delimited, optional) RemoveAliases
(semicolon-delimited, optional) PreserveX500 (optional, default true)
ValidateDomainExists (optional)

Behavior: - Normalize smtp/SMTP prefixes. - Prevent removing primary
without replacement. - Compute diff before applying changes. -
Idempotent operations (safe no-op where applicable).

40-Create-SPOSites.ps1

Purpose: Provision SharePoint Online sites from CSV.

Input CSV Required Headers: SiteUrl Title Template PrimaryOwnerUPN
SecondaryOwnersUPNs (optional) Language (optional) TimeZoneId (optional)
StorageQuotaMB (optional)

Behavior: - Validate URL format and tenant alignment. - Create site if
not existing. - Add secondary owners. - Safe no-op if site already
exists (unless fail flag specified).

41-Set-SPOSiteAdmins.ps1

Purpose: Set site collection administrators for targeted sites.

Input CSV Required Headers: SiteUrl AddSiteAdmins (semicolon-delimited)
RemoveSiteAdmins (optional) EnsurePrimaryOwnerIsAdmin (optional, default
true)

Behavior: - Idempotent add/remove. - Prevent removal of last admin
unless explicitly overridden.

42-Create-SPOHubSites.ps1

Purpose: Promote existing sites to Hub Sites.

Input CSV Required Headers: SiteUrl HubDisplayName (optional
informational) HubDescription (optional) HubLogoUrl (optional)

Behavior: - Confirm site exists. - Register as hub if not already. -
Safe no-op if already hub.

43-Associate-SPOSitesToHub.ps1

Purpose: Associate sites to a specific Hub Site.

Input CSV Required Headers: HubSiteUrl (or HubSiteId) SiteUrl
EnforceSameTenant (optional, default true)

Behavior: - Validate hub exists. - Associate site if not already
associated. - Support optional reassociation flag.

52-Add-TeamOwners.ps1

Purpose: Add owners to existing Teams.

Input CSV Required Headers: TeamId (preferred) or GroupId OwnerUPNs
(semicolon-delimited) EnsureOwnerCountMin (optional)

Behavior: - Resolve team and users. - Add as owners if not already. -
Ensure team never ends without owners. - Add-only script (no removals).

93-Compare-DesiredVsActual.ps1

Purpose: Generic diff engine comparing desired CSV to actual CSV.

Parameters: -DesiredCsvPath -ActualCsvPath -KeyColumns -CompareColumns
(optional) -OutputCsvPath

Behavior: - Normalize case and whitespace. - Normalize multi-value
semicolon fields. - Output differences as: MissingInActual ExtraInActual
Mismatch

Output Columns: DiffType Key ColumnName DesiredValue ActualValue Notes

End of Specification
