# OnPrem Folder

`OnPrem` is the planning and implementation area for on-premises workloads.

Current status: planning matrix is defined for ActiveDirectory (`00xx`), GroupPolicy (`01xx`), ExchangeOnPrem (`02xx`), and FileServices (`03xx`). No on-prem scripts are implemented yet.

## Planned Workloads

- `00xx`: ActiveDirectory
- `01xx`: GroupPolicy
- `02xx`: ExchangeOnPrem
- `03xx`: FileServices

## Operation Folders

- `OnPrem/Provision/`
- `OnPrem/Modify/`
- `OnPrem/InventoryAndReport/`

## Matrix Overview

| Operation | ActiveDirectory (`00xx`) | GroupPolicy (`01xx`) | ExchangeOnPrem (`02xx`) | FileServices (`03xx`) |
|---|---|---|---|---|
| Provision (`P`) | Planned | Planned | Planned | Planned (scope to be refined) |
| Modify (`M`) | Planned | Planned | Planned | Planned (scope to be refined) |
| InventoryAndReport (`IR`) | Planned | Planned | Planned | Planned (scope to be refined) |
