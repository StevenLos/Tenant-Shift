# Discover_OutputCsvPath

Default output folder for Discover scripts.

- Default result file pattern: `Results_IRWWNN-<Action>-<Target>_<yyyyMMdd-HHmmss>.csv`
- Required transcript file pattern: `Transcript_IRWWNN-<Action>-<Target>_<yyyyMMdd-HHmmss>.log`
- Override per run with `-OutputCsvPath`
- Script runs automatically write both result CSV and transcript log here by default
