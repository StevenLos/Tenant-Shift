# Common Folder

`Common` stores shared modules and helper assets.

## Structure

- `Common/Online/`: modules used by cloud workload scripts
- `Common/OnPrem/`: modules reserved for on-prem workload scripts
- `Common/Shared/`: cross-environment helper modules/assets

## Current Modules

- Online module: `./Common/Online/M365.Common.psm1`

On-prem and shared modules are scaffolded and ready for new workload implementation.
