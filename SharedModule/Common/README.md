# Common Folder

`Common` stores shared modules and helper assets.

## Structure

- `SharedModule/Common/Online/`: modules used by cloud workload scripts
- `SharedModule/Common/OnPrem/`: modules reserved for on-prem workload scripts
- `SharedModule/Common/Shared/`: cross-environment helper modules/assets

## Current Modules

- Online module: `./SharedModule/Common/Online/M365.Common.psm1`
- On-prem module: `./SharedModule/Common/OnPrem/OnPrem.Common.psm1`

Shared modules remain scaffolded and ready for future cross-environment implementation.
